NetGuard-IoT es un proyecto orientado a la supervisión y análisis de tráfico en redes IoT inalámbricas.
El sistema permite capturar tráfico generado por dispositivos ESP32, analizarlo en tiempo real y detectar comportamientos anómalos o dispositivos no autorizados dentro de la red.

El proyecto está dividido en tres módulos principales:
- Recolector de tráfico
- Analizador de anomalías
- Dashboard web y API
